# Installation

1. Paketdatei installieren:
   `pkg_stpageflip-2.0_full.zip` aus dem aktuellen Build-Ordner installieren.
2. Danach in Joomla prüfen, dass diese Plugins installiert sind:
   - `Content - stPageFlip`
   - `AJAX - ST PageFlip`
   - `Button - ST PageFlip`
3. Plugins aktivieren, falls Joomla sie nicht automatisch aktiviert.
4. Bücher unter `images/stpageflip/BUCHNAME/` im Joomla-Medienmanager anlegen.
5. Für PDF-Konvertierung sicherstellen:
   - PHP `imagick` aktiv
   - PDF-/Ghostscript-Support in Imagick vorhanden
   - Schreibrechte für `images/stpageflip/`
